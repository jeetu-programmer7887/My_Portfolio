# Cinematic Portfolio

A complete Next.js 14 cinematic developer portfolio with animated sections, custom cursor, terminal typewriter, count-up stats, project cards, timeline, contact form, and canvas particles.

## Run Locally

PowerShell may block the `npm` script shim on this machine, so use `npm.cmd`:

```bash
npm.cmd install
npm.cmd run dev
```

Then open `http://localhost:3000`.

## Customize

Replace `YOUR NAME`, `NAME.dev`, `email@domain.com`, and the social links with your own details. The About section includes a portrait placeholder ready to swap for a real image.
