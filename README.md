## 🚀 Overview

This is my **personal developer portfolio** — a production-grade web application built to showcase my skills, projects, and professional story as a Full Stack Developer. Hosted on a **custom domain** at [jeetuprasad.in](https://www.jeetuprasad.in), this portfolio reflects my commitment to crafting fast, accessible, and obsessively well-built digital experiences.

Built with **Next.js 14**, styled with **Tailwind CSS**, and enriched with smooth animations, the portfolio is designed to feel like magic while working like clockwork.

## 🚀 Getting Started

### Prerequisites
- Node.js `v18+`
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/jeetu-programmer7887/My_Portfolio.git

# Navigate into the project directory
cd My_Portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the portfolio locally.

### Build for Production

```bash
npm run build
npm start
```

## 🌐 Deployment

This portfolio is deployed on **Vercel** and connected to the custom domain **[www.jeetuprasad.in](https://www.jeetuprasad.in)**.

To deploy your own fork:

1. Push your code to GitHub
2. Import the repository on [Vercel](https://vercel.com)
3. Add your custom domain under **Project Settings → Domains**
4. Configure DNS records with your domain registrar
